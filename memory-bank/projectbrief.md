## Project Overview
This project is a modern, responsive dashboard for monitoring Ford vehicle fleets, developed as an Angular frontend application with a Node.js/Express backend API. The system provides fleet management capabilities including vehicle monitoring, authentication, and VIN-based vehicle data retrieval.

## Core Objectives
- Create an interactive dashboard displaying vehicle fleet metrics and status
- Implement secure user authentication system
- Provide vehicle catalog with technical data and images
- Enable VIN-based search with real-time location and status information
- Ensure responsive design for desktop and mobile devices

## Key Features
1. **Dashboard**: Interactive metrics visualization for fleet monitoring
2. **Authentication**: Secure login system with user management
3. **Vehicle Catalog**: Display of Ford vehicle models with specifications
4. **VIN Search**: Detailed vehicle lookup by Vehicle Identification Number
5. **Responsive UI**: Modern, professional interface across all devices

## Technical Scope
- **Frontend**: Angular 19+ SPA with TypeScript
- **Backend**: Node.js/Express REST API
- **Data**: Mock data via JSON Server for development
- **Deployment**: Hybrid architecture (Vercel frontend + Render backend)

## Success Criteria
- Functional authentication system
- Complete vehicle data display and search
- Responsive design working on all screen sizes
- Clean, maintainable codebase following Angular best practices
- Successful deployment to production environment

## Constraints
- Must use specified technology stack (Angular + Node.js)
- Should follow Ford branding and vehicle data structure
- Must be deployable to Vercel/Render architecture
- Should maintain separation between frontend and backend concerns
